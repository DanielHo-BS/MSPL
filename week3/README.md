# Usage

## Environment

**``Anaconda`` + ``jupyter notebook`` (or ``VScode``).**

## Build

Get the code.

```bash
cd ~
git clone git@github.com:DanielHo-BS/MSPL.git
cd MSPL/week3
```

## Install packages
```bash
pip install -r requirements.txt
```

## Run
```bash
python3 main.py
```
